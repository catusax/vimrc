### 引用
https://github.com/bramblex/BlxVimrc

### 安装
```
rm ~/.vim ~/.vimrc -rf
git clone https://github.com/catusax/vimrc.git ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
进入Vim，安装插件
```
:BundleInstall
```

### 快捷键
F5 粘贴模式
F4 历史记录
